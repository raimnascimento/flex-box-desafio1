# Desafio 1 - Flexbox

Este repositório contém a resolução do Desafio 1 de Flexbox, realizado em aula na disciplina de PI.

Para resolver o desafio, foram utilizados:

* `justify-content` para deixar os elementos espaçados
* `align-self` para alinhar individualmente os itens dentro do container
* `order` para definir a ordem de cada elemento

O objetivo da atividade foi praticar o posicionamento de elementos utilizando Flexbox.
